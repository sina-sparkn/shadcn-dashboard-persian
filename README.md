A modern, responsive, and feature-rich admin dashboard template built with Next.js 14+ (App Router), TypeScript, Tailwind CSS, and shadcn/ui components.[1] Designed for developer experience and performance.[2][3]

![alt text](/public/image.png)

üöÄ Features

Framework: Built with Next.js 14+ (App Router) for server-side rendering and static site generation.

UI Components: Beautifully designed components from shadcn/ui.[1][2][4]

Styling: Tailwind CSS for utility-first styling.

Type Safety: Fully written in TypeScript.

Authentication: Integrated with [NextAuth.js / Clerk / Supabase] (Edit as needed).

Charts: Interactive data visualization using Recharts.

Dark Mode: Built-in light and dark mode toggle.

üé® Customization
Changing the Theme

This template uses Shadcn UI's theming system.[2][3][6][7] You can easily customize the primary colors, border radius, and more.

Go to app/globals.css.[2]

Update the HSL color variables in the :root and .dark sections.

Alternatively, use the shadcn themes page to generate new CSS variables and copy them into your file.
